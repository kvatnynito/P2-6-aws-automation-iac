# Screenshots

This directory will store redacted screenshots for the AWS Automation & Infrastructure as Code project.

## Screenshot Rules

Before uploading screenshots, verify that they do not expose sensitive information.

Do not commit screenshots showing:

- AWS account ID
- Access keys
- Secret access keys
- Session tokens
- Private SSH keys
- Billing details
- Personal email addresses
- Unredacted ARNs if sensitive
- Terraform state content
- Sensitive command output

## Planned Screenshot Evidence

Potential screenshots may include:

- AWS CLI identity validation
- AWS CLI resource query output
- Bash script output
- Python/boto3 script output
- Terraform plan output
- Terraform apply output
- Terraform destroy output
- Cleanup validation

## Redaction Requirement

All screenshots must be reviewed and redacted before being committed.
