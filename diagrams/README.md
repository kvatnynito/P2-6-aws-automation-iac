# Diagrams

This directory will store diagrams for the AWS Automation & Infrastructure as Code project.

## Planned Diagrams

Potential diagrams may include:

- AWS automation workflow
- AWS CLI validation workflow
- Python/boto3 security check workflow
- Terraform deployment workflow
- Cleanup automation workflow

## Diagram Requirements

Diagrams should clearly show:

- Input
- Script or tool
- AWS service queried or modified
- Output or finding
- Review step
- Cleanup step

## Security Notes

Diagrams should not expose sensitive information.

Do not include:

- AWS account ID
- Access keys
- Secret keys
- Session tokens
- Private SSH keys
- Personal email addresses
- Billing information
- Unredacted public IPs unless intentionally documented
- Sensitive resource identifiers
