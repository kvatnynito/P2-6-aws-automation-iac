# Bash Scripts

This directory will store Bash scripts for AWS automation.

## Planned Bash Scripts

Potential scripts may include:

- `check-identity.sh`
- `list-running-ec2.sh`
- `list-s3-buckets.sh`
- `review-security-groups.sh`
- `cleanup-check.sh`

## Bash Script Rules

Scripts should:

- Use clear comments
- Print what they are doing
- Avoid hardcoded credentials
- Require confirmation before destructive actions
- Handle errors where possible
- Be safe to run in a lab environment
